# True Future Frontend

진짜 미래 확인하기 프론트엔드 애플리케이션

## 설치 방법

```bash
cd frontend
npm install
```

## 실행 방법

개발 서버 실행:
```bash
npm run dev
```

브라우저에서 `http://localhost:5173` (또는 터미널에 표시된 주소)로 접속하세요.

## 빌드 방법

프로덕션 빌드:
```bash
npm run build
```

빌드 결과 미리보기:
```bash
npm run preview
```

## 사용 기술

- React 18
- Vite
- Tailwind CSS
